## QQ Music using React

##### Imitating QQ Music  using  React and Ant Design

### 如何启动项目

### 主要功能

#### 1.首页

* 关闭广告
* 导航切换(首页、热歌榜、搜索)
* Carousel 插件显示轮播图
* 每日推荐: 显示专辑封面和专辑描述，点击跳转至播放列表
* 新歌速递: 显示新歌名字、专辑与歌手，点击跳转至播放页面(List组件)

#### 2.热歌榜

* 显示封面图片、标题、最后更新时间
* 显示歌单列表、专辑与歌手，点击跳转至播放页面
